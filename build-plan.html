Section 01
What Things Actually Cost
The Honest Truth

The previous version had setup costs of £1,500–£5,000 for simple modules. That was wrong — it assumed you were paying a developer. You're not. Claude is your developer. The real cash cost of building most modules is £100–500 in API calls, existing subscriptions, and VA time. The investment is your time — not your cash.

✓ What You're Actually Spending Cash On
  • Claude API during the build — £20–80 per module
  • Twilio setup + test messages — £20–50 one-time
  • Apify scraper credits — £40–80/mo for intel modules
  • VA deployment time — £80–150 per module
  • Domain names — £10–15/year per vertical brand
  • GHL, Make.com, Vercel — shared across all modules, not per module
  • Compliance review (clinical/regulated only) — £500–800 once
  • Industrial specialist (FactoryOS sensors only) — £8–20k. The only big external cost.
✕ What You Are NOT Paying For
  • Developer fees — Claude writes all code. £0.
  • Designer fees — Claude builds the UI. £0.
  • DevOps / deployment fees — Vercel + VA following instructions. £0.
  • Database admin — Supabase/Neon managed. Claude configures. £0.
  • Agency fees — no agencies. You and AI do the work. £0.
  • Project management tools — Claude + free Notion. £0.
  • Hardware (mostly) — almost everything is browser and cloud. Exception: FactoryOS sensor layer.
The Real Investment

Stage 1 modules cost your time, not your cash. Each simple module takes 20–60 hours of working sessions with Claude over 1–3 weeks. The question is not "can we afford to build this" — it is "do we have the focused hours right now."

Why Staging Matters

You don't need to build everything before you can sell. Build the one module that creates the most obvious ROI for your first client. Sign that client. Use their revenue to fund the next module. Every vertical has a Stage 1 MVP under £500 in real cash, live within 3 weeks. Everything else follows from there.

Section 03
The AI Build Toolchain
Claude (Anthropic)
Writes all code. Dashboards, APIs, databases, integrations, automation logic. Your primary developer. 80% of build work happens here.
Perplexity AI
Research. API docs, integration guides, pricing, competitor analysis. Use at the start of every module build to understand what you're building into.
n8n (self-hosted)
Visual workflow automation. Connects apps, schedules tasks, triggers AI. ~£10/mo VPS. Free software. No code to operate once built.
Make.com
Cloud automation. Easier for non-technical team. Client-facing comms — SMS sequences, email triggers, CRM updates. £29–£99/mo.
Vercel
Deploys all apps and dashboards. One-click from GitHub. Free tier covers early stage. VA follows Claude's instructions. No server knowledge needed.
Supabase / Neon
Database and auth. Stores all client data. Free tiers cover early stage. Claude configures everything — you just manage the dashboard.
GoHighLevel (GHL)
CRM, email, SMS, funnels. Client-facing comms layer for Dental and Health. One agency account (£297/mo) = unlimited client sub-accounts.
Apify
Web scraping. Powers competitor intelligence — pulls pricing, reviews, market data automatically. No coding needed to run scrapers once built.
Twilio
SMS and WhatsApp API. ~1p per SMS. Pay-as-you-go. No monthly fee to start. Claude integrates it.
Section 04
How Every Module Gets Built
01
Define & Research
Perplexity + Claude
Describe what the module does. Research APIs and integrations. Claude helps structure a spec before any code is written.
02
Build with Claude
Claude
Give Claude the spec. It writes the full app — frontend, backend, database, APIs. Iterate until working. Days, not months.
03
Automate
n8n / Make.com
Build the automation layer — triggers, schedules, messages, syncs. Claude writes the logic; n8n/Make runs it. No code needed.
04
Deploy
Vercel + GitHub
Claude generates step-by-step instructions. VA follows them — push to GitHub, connect Vercel, set env vars. Live in under an hour.
05
Monitor
Vercel + Supabase
Vercel watches uptime. Paste any errors into Claude for instant fixes. Most modules need under 2 hours/month once live.
